## 2の補数の計算方法

数の2の補数を計算するには、次の手順を実行できます。

1. 数字を2進数形式で表現します。
2. その数の補数を見つけます。補数は、各ビットの0を1に、1を0に変えることで得られます。
3. 補数に1を加えます。

例えば、数値-5の2の補数を計算したい場合：

1. -5を2進数に変換し、11111011が得られます。
2. 補数を見つけ、各ビットを0を1に、1を0に変えると、00000100が得られます。
3. 補数に1を加えると、00000101が得られ、これが数値-5の2の補数です。

### JavaScriptを使用した2の補数の計算のサンプルプログラム

以下はJavaScriptを使用して数の2の補数を計算するサンプルプログラムです。

```javascript
// 数の2の補数を計算
function calculateTwosComplement(number) {
  // 数字を2進数文字列に変換
  const binary = number.toString(2);

  // 補数を計算
  let complement = "";
  for (let i = 0; i < binary.length; i++) {
    complement += binary[i] === "0" ? "1" : "0";
  }

  // 補数を数字に変換し、1を加えて2の補数を得る
  return parseInt(complement, 2) + 1;
}

// 使用例
const number = -5;
const twosComplement = calculateTwosComplement(number);

console.log("数値", number, "の2の補数は", twosComplement);
```

上記のサンプルプログラムを使用すると、数値を入力してその2の補数を計算することができます。
