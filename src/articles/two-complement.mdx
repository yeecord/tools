## 什麼是二的補數？

在計算機科學中，二的補數是一種表示負數的方法。二的補數可以將負數轉換為正數，並使計算機更容易處理這些數字。二的補數通常由一個正數的補數加上 1 來獲得。例如，對於一個 8 位元的二進制數，將一個正數的補數加上 1 可以得到該數的二的補數。

## 計算二的補數的方法

計算一個數的二的補數可以通過以下步驟進行：

1. 將數字表示為二進制形式。
2. 找到該數的補數。補數可以通過將每個位元的 0 變為 1，1 變為 0 來獲得。
3. 將補數加上 1。

舉例來說，我們想要計算數字 -5 的二的補數：

1. 將 -5 轉換為二進制形式，得到 11111011。
2. 找到補數，將每個位元的 0 變為 1，1 變為 0，得到 00000100。
3. 將補數加上 1，得到 00000101，這就是數字 -5 的二的補數。

### 使用 JavaScript 計算二的補數的範例程式

下面是使用 JavaScript 計算一個數的二的補數的範例程式：

```javascript
// 計算數字的二的補數
function calculateTwosComplement(number) {
  // 將數字轉換為二進制字串
  const binary = number.toString(2);

  // 計算補數
  let complement = "";
  for (let i = 0; i < binary.length; i++) {
    complement += binary[i] === "0" ? "1" : "0";
  }

  // 將補數轉換為數字並加上 1 得到二的補數
  return parseInt(complement, 2) + 1;
}

// 使用範例
const number = -5;
const twosComplement = calculateTwosComplement(number);

console.log("數字", number, "的二的補數是", twosComplement);
```

使用上述範例程式，我們可以輸入一個數字並計算其二的補數。
